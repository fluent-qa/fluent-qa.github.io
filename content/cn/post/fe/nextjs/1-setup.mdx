## Setup Nextjs Project from Scratch

可以使用 Bun、TailwindCSS 和 TypeScript 创建一个 Next.js 项目，以下是步骤：

1. **使用 Bun 初始化项目**

   Bun 是一款快速且强大的 JavaScript 运行时，可以用来创建 Next.js 项目。首先，使用 Bun 的 `bunx` 命令创建一个新的 Next.js 项目：

   ```bash
   bunx create-next-app@latest my-nextjs-app --ts
   ```

   这将使用 Bun 创建一个名为 `my-nextjs-app` 的新 Next.js 项目，并使用 TypeScript 作为语言。

2. **安装 TailwindCSS**

   接下来，安装 TailwindCSS 和它的依赖：

   ```bash
   bunx add tailwindcss postcss autoprefixer
   ```

3. **配置 TailwindCSS**

   创建 `tailwind.config.js` 文件，并在其中定义 TailwindCSS 的配置选项：

   ```javascript
   /** @type {import('tailwindcss').Config} */
   module.exports = {
     content: [
       './pages/**/*.{js,ts,jsx,tsx}',
       './components/**/*.{js,ts,jsx,tsx}',
     ],
     theme: {
       extend: {},
     },
     plugins: [],
   }
   ```

   然后，创建一个名为 `globals.css` 的 CSS 文件，并在其中添加以下内容：

   ```css
   @tailwind base;
   @tailwind components;
   @tailwind utilities;
   ```

4. **设置 Bun 的 TypeScript 配置**

   在项目的根目录下创建 `tsconfig.json` 文件，并在其中添加以下配置：

   ```json
   {
     "compilerOptions": {
       "target": "esnext",
       "module": "esnext",
       "moduleResolution": "node",
       "strict": true,
       "jsx": "preserve",
       "esModuleInterop": true,
       "allowSyntheticDefaultImports": true,
       "forceConsistentCasingInFileNames": true,
       "noImplicitAny": true,
       "noImplicitThis": true,
       "noUnusedLocals": true,
       "noUnusedParameters": true,
       "strictNullChecks": true,
       "skipLibCheck": true,
       "outDir": "dist",
       "baseUrl": ".",
       "paths": {
         "@/*": ["./src/*"]
       }
     },
     "include": [
       "next.config.js",
       "pages/**/*.{ts,tsx}",
       "components/**/*.{ts,tsx}",
       "src/**/*.{ts,tsx}"
     ],
     "exclude": [
       "node_modules"
     ]
   }
   ```

   这将告诉 Bun 如何编译 TypeScript 代码。

5. **开始开发**

   现在，你就可以开始使用 Bun、TailwindCSS 和 TypeScript 开发你的 Next.js 项目了。可以使用 `bunx dev` 命令启动开发服务器：

   ```bash
   bunx dev
   ```

   Bun 将会自动检测代码更改并重新加载页面。

现在，你已经成功地使用 Bun、TailwindCSS 和 TypeScript 创建了一个 Next.js 项目，并可以开始开发你的应用程序了。[5](https://tailwindcss.com/docs/guides/nextjs)[6](https://nextjs.org/docs/app/building-your-application/styling/tailwind-css)
